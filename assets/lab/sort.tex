\subsection*{Ordinamento (\texttt{mergeSort})}

Dato un array di interi, ordinatelo. Siete liberi di utilizzare le librerie, ma implementare un algoritmo di ordinamento
visto a lezione può essere un esercizio più utile.

\paragraph{Formato dell'input}
La prima riga del file di input contiene \(N\), il \emph{numero di interi nell’array}. La seconda riga contiene l’array, con gli elementi separati
da spazi.

\paragraph{Formato dell'output}
L'output deve contenere gli elementi ordinati dell’array, separati da uno spazio.
